# complete — Kitchen-Sink Reference

This is the comprehensive reference config showing all baker features together:
merge, pack, convert, custom OTA header DSL, version extraction, and groups.

For focused single-feature examples, see the sibling directories.

## Run

```bash
baker build                  # default target (merge_test)
baker build pack_bin         # OTA pack
baker build convert_bin      # convert to bin
baker build convert_srec     # convert to srec
```
